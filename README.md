# SegmentFinder
多进程C段Web扫描器

识别目标网段内存活Web主机并扫描常见HTTP端口，自动整理输出为文件的简单脚本


主要用于梳理目标网段内的Web资产，并配合指纹识别工具寻找出重点资产


被设计为主要用于结合Ehole或Finger使用，该工具输出结果用来直接读取至Ehole进行指纹识别


扫描时会输出title信息方便手工整理重要资产，扫描结束后会自动生成结果文件（outfile.txt）
